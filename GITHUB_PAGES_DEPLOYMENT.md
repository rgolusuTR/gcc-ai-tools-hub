# GitHub Pages Deployment Guide

## 🚀 Deploy Your GCC AI Tools Hub to GitHub Pages

Follow these simple steps to make your application live on the internet for FREE!

---

## Step-by-Step Instructions

### Step 1: Sign in to GitHub
1. Open your browser and go to: **https://github.com**
2. Click **"Sign in"** (top right)
3. Enter your credentials

### Step 2: Navigate to Your Repository
1. Go to: **https://github.com/rgolusuTR/gcc-ai-tools-hub**
2. You should see your repository with all the files

### Step 3: Enable GitHub Pages
1. Click on **"Settings"** tab (top menu, near the right)
2. In the left sidebar, scroll down and click **"Pages"**
3. Under **"Source"**, you'll see a dropdown that says "None"
4. Click the dropdown and select **"main"** (or "master" if that's your branch name)
5. Leave the folder as **"/ (root)"**
6. Click **"Save"**

### Step 4: Wait for Deployment (1-2 minutes)
1. GitHub will show a message: "Your site is ready to be published"
2. Refresh the page after 1-2 minutes
3. You'll see a green box with: "Your site is published at https://rgolusutr.github.io/gcc-ai-tools-hub/"

### Step 5: Access Your Live Site! 🎉
Your application is now live at:
```
https://rgolusutr.github.io/gcc-ai-tools-hub/
```

## 🔧 Troubleshooting

### Issue: 404 Page Not Found
**Solution**: Wait a few more minutes. GitHub Pages can take up to 10 minutes for the first deployment.

### Issue: Page shows old content
**Solution**: 
1. Clear your browser cache (Ctrl + Shift + Delete)
2. Or try opening in an incognito/private window

### Issue: Can't find "Pages" in Settings
**Solution**: 
1. Make sure you're signed in
2. Make sure you're in the repository settings (not your account settings)
3. Scroll down in the left sidebar - "Pages" is near the bottom

## 🔄 Updating Your Site

Whenever you make changes to `index.html`:

1. Save your changes
2. Commit: `git add index.html && git commit -m "Update content"`
3. Push: `git push origin main`
4. Wait 1-2 minutes
5. Refresh your live site - changes will appear!

## 🎯 Custom Domain (Optional)

If you want a custom domain like `tools.yourcompany.com`:

1. In GitHub Pages settings, find "Custom domain"
2. Enter your domain name
3. Follow GitHub's DNS configuration instructions
4. Wait for DNS propagation (can take 24-48 hours)
